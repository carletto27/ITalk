1) Utilizzare preferibilmente VisualStudio2019 per eseguire il programma
2) E' stato utilizzato il MYSQL Workbench
3) Ricordare di modificare username e password all'interno del ServerCPPNet e di ProgettoAPLAdmin per interagire con il database
4) All'interno del modulo progettoAPL1 (client), nel modulo WrMsg riga 47 inserire un path valido per la creazione del file
5) Aggiungere Newtonsoft.Json NuGet package, in modo tale da utilizzare i json nel client ( https://moredvikas.wordpress.com/2019/03/06/add-the-newtonsoft-json-nuget-package-in-visual-studio-to-dot-net-project/ )

•	Server: ServerCPPNet
•	Client: ProgettoAPL1
•	Admin e Dati statistici: ProgettoAPLAdmin
